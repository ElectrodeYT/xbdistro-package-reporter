API base URL

This is the base URL for the xbdistro-package-reporter API. This instance is configured with distro_name = {{ distro_name }}, and allow_email_config = {{ allow_email_config }}.

The API is not yet stable.

/email

All URLs under /email return human-readable HTML. There may be changes at any time here. Responses are not designed to be machine-interpreted.

/packages

/packages/list

Returns a list of packages.

/packages/package/<name>

Returns more information about a specific package.
